이 사이트는 별도의 프레임워크나 빌드 도구 없이 순수 **HTML, CSS, JavaScript**로 만들어졌습니다. `posts/` 폴더에 마크다운 파일을 추가하고 `posts.json`에 파일명을 등록하면 자동으로 목록에 나타납니다.

## 지원하는 문법

- **굵게**, *기울임*, `인라인 코드`
- [링크](https://example.com)
- 순서 없는 목록과 순서 있는 목록
- 인용문
- 코드 블록

> 마크다운은 읽기도 쓰기도 쉬운 문서 형식입니다.

```js
function greet(name) {
  return `Hello, ${name}!`;
}
```

앞으로 여기에 글을 하나씩 쌓아갈 예정입니다.
