## 왜 프레임워크 없이 만들었나

빌드 도구가 없으면 배포가 단순해집니다. HTML, CSS, JS 파일 몇 개와 마크다운 글 파일만 있으면 됩니다.

1. `posts/*.md` 파일에 글을 작성한다.
2. `posts/posts.json`에 파일명을 추가한다.
3. 정적 서버에 올린다.

이게 전부입니다. 별도의 `npm install`이나 빌드 단계가 필요 없습니다.

## 다크 모드는 어떻게 동작하나

`data-theme` 속성을 `<html>` 태그에 두고 CSS 커스텀 프로퍼티로 색상을 전환합니다. 저장된 값이 없으면 시스템 설정(`prefers-color-scheme`)을 따릅니다.
